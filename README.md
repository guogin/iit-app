# IIT App

A simple Individual Income Tax (IIT) calculator for China. It accepts annual income, bonuses, and deductions, supports English and Chinese UI, and sends the inputs to the backend simulator.

![Screenshot](screenshot.png)

## Features

- Multiple income categories and deductions
- Expression input for numeric fields (for example: 18000 * 12)
- Dual language support (zh-CN and en-US)
- One-time and integrated taxation results with trace logs

## Run Locally

1) Install dependencies

```bash
npm install
```

2) Start the dev server

```bash
npm start
```

Open http://localhost:3000 in your browser.

## Build

```bash
npm run build
```

## Test

```bash
npm test
```
